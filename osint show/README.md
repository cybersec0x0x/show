# Project Catarsis Desktop

Large desktop workstation for:

- public-source OSINT
- cyber intel and IOC lookup
- open datasets and registries
- maps, media, archives, and domain research
- collections, case notes, and creator branding

## Main sections

- Dashboard
- Search Hub
- Resource Atlas
- Intel Tools
- Collections
- Studio
- Show
- Caseboard

## Creator

- Alias: `show`
- Discord: `h2sv`

## Included upgrades

- fixed menu edge styling
- reliable external link opening through Electron IPC
- theme and font studio
- density and effect toggles
- search history and recent launches
- bookmark collections
- case export and import
- Windows `.exe` build configuration

## Run

```powershell
npm.cmd install
npm.cmd start
```

## Build Windows exe

```powershell
npm.cmd run build:win
```
