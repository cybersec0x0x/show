Mission Control

Project Catarsis is now a desktop command center with broad public-source coverage and a cleaner visual shell.

This build focuses on real workflow: search presets, public registries, cyber intel tools, visual resource atlas, settings studio, collections, recent launches, import/export, and creator presence.

Feature Deck

  • Category-based public search routing for people, company, cyber, domain, media, maps and archives.
  • Large atlas of open resources with filters, bookmarks and fast launch.
  • Theme studio with multiple themes, fonts, density modes and effect toggles.
  • Recent launches, search history, export and import for collections and cases.
  • Dedicated creator page for show and Discord contact h2sv.

Visual Fixes

White menu edges were removed by resetting native button appearance, softening border strokes, replacing hard panel separators, and adding theme controls for border-free layouts.
